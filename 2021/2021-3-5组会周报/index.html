2021-3-5组会周报

组会 2021-03-04 739 字 2 分钟 次

本周工作

  1. 阅读论文：

    题目 来源 解决问题
    Embedding Temporal Network via Neighborhood Formation KDD2018 最早将霍克斯过程带入到时序网络嵌入问题当中的论文，提出了HTNE算法，加入了历史时间对现在发生事件的正向激励
    Temporal Network Embedding with Micro- and Macro-dynamics CIKM2019 加入了宏观节点之间的边的数量约束，改进了HTNE，本质上依旧是霍克斯过程
    Dynamic Network Embedding byModeling Triadic Closure Process AAAI 2018 加入了triad的概念，将网络信息的传播过程看作由open triad变为closed triad的过程。同样利用霍克斯过程对时序网络中的节点进行嵌入

    目前已经跑通HTNE算法，后续的改进主要有以下几点：

    1. 加入motif（三个节点，两条边）的影响，由于IJCAI20那篇文章（MTNE），只考虑了open triad变成closed triad的过程，本身open triad 自身是由两条边构成的，这两条边之间也存在者相互影响的关系，可以考虑把这部分影响加入。

    2. 由于MTNE只考虑正向影响，因此可以在条件强度函数中加入负向影响

    3. 在loss中加入对比学习（互信息最大化），使得生成的embedding效果更好，因为对比学习本身具有聚类的效果，可以拉近相似节点embedding的距离，拉大不同节点embedding的距离。但是这里涉及到负采样策略的问题，如何生成负样本这部分如何改进还在考虑。

    4. 历史节点不更新，由于霍克斯过程的公式中有定义历史影响，而HTNE的代码中同时更新与发生事件相关的两个节点和历史节点，后续看一下是历史节点不更新效果会不会好一点

  2. 调研区块链的平台和技术：

    目前区块链中比较成熟的技术就是比特币和以太坊，而比特币仅仅支持数字货币，以太坊是一个开源的有智能合约功能的公共区块链平台，是目前使用最广泛的支持完备应用开发的公有区块链系统，可支持开发人员自定义各种脚本，所以还是选择采用以太坊技术。后面会在几台节点上搭一个以太坊的子链。
