强化学习论文总结

强化学习相关 2020-08-13 767 字 2 分钟 次

##由于研究生期间主要是做强化学习相关内容，所以取Google Scholar 找了若干篇高被引论文，希望可以快速入门强化学习领域。本文主要记录文章的相关内容以及缺点。

Finding key players in complex networks through deep reinforcement

年份：2020

DIO：$10.1038/s42256-020-0177-2​$

发表期刊：Nature Machine Intelligence

  • 主要研究问题：利用Reinforcement Learning在网络中寻找一些关键节点
  • 研究动机（Motivation）：针对目前在网络中寻找关键节点的方法都是启发式的，即在一定条件下才可以使用，没有一种通用的方法来解决这个问题。所以这篇文章提出了一个通用的框架
  • 主要方法：提出了一个__FINDER__框架，利用图嵌入方式，为每个网络生成一个Embedding，在结合Q-Learning实现关键节点的寻找
  • 缺点：
    1. 不适用于大规模（NP-Hard）
    2. 不适用于动态图，文中做实验用到的基本是静态图

A novel reinforcement learning algorithm for virtual network embedding

年份：2018

DIO：10.1016/j.neucom.2018.01.025 ​$

发表期刊：Neurocomputing（C类，SCI三区）

  • 主要研究问题：关于虚拟网络的嵌入问题，即给一个虚拟的请求，从一系列结点之间传递数据，要在实际的网络中建立物理节点与虚拟结点之间的映射。
  • 研究动机（Motivation）：传统的虚拟网络嵌入方法都是启发式的，利用历史请求数据来训练RL以前打方法没有用过。
  • 主要方法：利用结点资源的CPU计算能力，链路的带宽，请求的持续时间，来定义Reward。据此来进行RL，此外由于action不是选择一个结点，而是要选择一条传输的路径。所以这篇文章采用了堆叠gradient的方式，每次根据输出的action计算gradient，然后将n次gradient都乘上reward和学习率作为最终的reward。
